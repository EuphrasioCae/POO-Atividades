# Programação Orientada a Objetos (POO)

A **Programação Orientada a Objetos (POO)** é um paradigma de programação baseado na organização do código em **objetos**, que representam entidades do mundo real e combinam **dados (atributos)** e **comportamentos (métodos)**.

Seus principais pilares são:

- **Encapsulamento**
- **Herança**
- **Polimorfismo**
- **Abstração**

Esses conceitos ajudam a tornar o código mais **modular**, **reutilizável** e **fácil de manter**.

Este repositório contém exemplos e práticas voltadas ao aprendizado e aplicação dos conceitos de **POO**.
